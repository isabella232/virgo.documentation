Current Milestone

This section lists important issues with the current Milestone release, e.g. M2.

Version Compatibility

The milestone does not work well with the current milestone of the runtime environment ("3.1.0" / 3.5.0). Please target the 3.0.2 Release Virgo Runtime or use the current snapshot build.
